How to play?

Trading Overview:
1 minute, 1 issue; 55 seconds to order; 5 seconds waiting for the draw. It opens all day. The total number of trades is 1440 issues.

Trade Cost:
If you spend $100 to trade, after deducting a service fee of 2%, the contract amount is $98.

Trade Options:
1. Select green: If the result shows 1, 3, 7, 9, you will get $196. If the result shows 5, you will get $147.
2. Select red: If the result shows 2, 4, 6, 8, you will get $196. If the result shows 0, you will get $147.
3. Select violet: If the result shows 0 or 5, you will get $441.
4. Select a specific number: If the result is the same as the number you selected, you will get $882.
5. Select big: If the result shows 5, 6, 7, 8, 9, you will get $196.
6. Select small: If the result shows 0, 1, 2, 3, 4, you will get $196.
